## 安装(先装node)
> npm install -g hexo-cli

## 设置全局
> ln -s /usr/src/node-v8.9.1-linux-x64/bin/hexo /usr/local/bin/hexo

## hello blog
> hexo init blog
> cd blog
> npm install
> hexo server

## 安装主题
> git clone https://github.com/iissnan/hexo-theme-next themes/next

## 后台启动
> nginx //启动nginx
> nohup hexo server & //nohup启动 logout
